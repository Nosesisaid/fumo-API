# fumo rest API
## Issues
If you find a repeated fumo please create an issue
## Objects

### Fumo

```json
{
  "_id": "fumo id",
  "URL": "fumo image url",
  "__v": 0
}
```

## Paths

### <code>/random</code>

return a random fumo object.

### <code> /fumos</code>

fumo list.

#### <code>/</code>

retrun the full fumo list.

#### <code>/id</code>

return the fumo object whit the provided id.
