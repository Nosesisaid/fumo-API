# fumo rest API
## Issues
If you find a repeated fumo please create an issue
## Objects

### Fumo

```json
{
  "_id": "fumo id",
  "URL": "fumo image url",
  "__v": 0
}
```

## Paths

### <code>/random</code>

Return a random fumo object.

### <code> /fumos</code>

Fumo list.

#### <code>/</code>

Return the full fumo list.

#### <code>/id</code>

Return the fumo object with the provided ID.
